The world's most verbose1 hello world web page

This is an example of using web-commonjs to load modules dynamically.

This example requires a web server, because web browsers do not allow web pages to load content (via XMLHttpRequest) when using the file:// protocol.

The page begins by requiring 'example/website', which loads website.js from the example directory. That module requires 'example/greetings', which in turn requires './messages', which is a relative import that resolves to 'example/messages'.

The 'example/website' module also imports jQuery, to demonstrate 'shimming' an existing global object into the module system as if it were a CommonJS module.
